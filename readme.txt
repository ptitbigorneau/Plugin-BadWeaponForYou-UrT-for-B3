# badweaponforyou plugin
# Plugin for B3 (www.bigbrotherbot.net)
# www.ptitbigorneau.fr

badweaponforyou plugin (v1.5.3) for B3

Installation:

1. Place the badweaponforyou.py in your ../b3/extplugins and the 
badweaponforyou.ini in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

<plugin name="badweaponforyou" config="@b3/extplugins/conf/badweaponforyou.ini"/>
